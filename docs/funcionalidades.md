# Funcionalidades

- **Scan Passivo:** Analisa o tráfego em busca de vulnerabilidades como headers de segurança ausentes, cookies inseguros e informações sensíveis.
- **Scan Ativo:** Envia payloads maliciosos para testar ativamente vulnerabilidades como SQL Injection, XSS e Command Injection.
- **Sender:** Permite enviar múltiplas requisições a partir de um arquivo de texto, substituindo um parâmetro dinamicamente.
- **Repetição (Repeater):** Permite reenviar uma requisição individual, com a possibilidade de modificar manualmente seus parâmetros e headers.
- **Decodificador (Decoder):** Ferramenta para codificar e decodificar dados em formatos como Base64, URL, HTML e Hex, além de gerar hashes.
- **Spider:** Navega automaticamente em um site para descobrir novas URLs, endpoints e formulários.
- **Editor de JWT:** Permite decodificar, modificar e assinar tokens JSON Web Tokens (JWT) para testar vulnerabilidades.
- **Cookie Jar:** Armazena e gerencia cookies capturados, permitindo forçar o uso de sessões específicas no Repeater e Sender.
- **Interceptação:** Captura e permite modificar requisições e respostas HTTP/HTTPS em tempo real.
- **Comparador (Comparator):** Ferramenta para comparar duas requisições ou respostas e destacar as diferenças.
- **Regras (Rules):** Permite criar regras para interceptar, modificar ou descartar tráfego com base em critérios específicos.
- **Tecnologias (Technologies):** Detecta e exibe as tecnologias (frameworks, servidores) usadas pelas aplicações.
- **WebSocket:** Permite visualizar e interceptar o tráfego de WebSockets.
- **Histórico (History):** Mantém um registro de todo o tráfego que passa pelo proxy para análise.
